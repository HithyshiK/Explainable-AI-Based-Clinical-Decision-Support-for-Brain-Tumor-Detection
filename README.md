# Explainable-AI-Based-Clinical-Decision-Support-for-Brain-Tumor-Detection
Developed a deep learning framework using EfficientNetV2B0 with Multi-Head Self-Attention for brain tumor classification. Integrated Explainable AI using Grad-CAM visualizations. Built AI-powered clinical decision support with downloadable medical reports.
